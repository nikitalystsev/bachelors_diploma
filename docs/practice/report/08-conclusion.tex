\chapter*{ЗАКЛЮЧЕНИЕ}
\addcontentsline{toc}{chapter}{ЗАКЛЮЧЕНИЕ}

Был реализован модуль модификации поведения робота Ф-2 на основе персональных черт.
Было проведено функциональное тестирование разработанного программного обеспечения.
Все тесты были успешно пройдены.
Также была описана цель проводимого исследования и критерии оценки качества работы метода.

Цель преддипломной практики была достигнута.
