## Слайд 8. Пример модификации реакции робота предлагаемым методом

На слайде показан пример работы модификации. 
Роботу на вход подается предложение.
На сервере робота это высказывание анализируется и формируется набор активированных д-сценариев и их весов.
В зависимости от заданных персональных черт (граней) путем перевзвешивания может меняться приоритетная реакция.

## Слайд 9. Формализация задачи

На вход модуля поступает набор активированных д-сценариев с исходными весами. 
Для каждого сценария вычисляется множитель, после чего вес пересчитывается как произведение исходного веса на этот множитель.
На выходе получается тот же набор сценариев, но уже с изменёнными весами.

## Слайд 10. Детализация модуля A3

На слайде представлена детализация блока А3: он получает активированные д-сценарии и, применяя управляющие воздействия, вычисляет коэффициенты перевзвешивания.
Далее полученные коэффициенты применяются для перевзвешивания.

## Слайд 11. Параметры предлагаемого метода

Параметры метода разделены на три группы: профиль персональных черт робота, параметры отношения к собеседнику и параметры текущей ситуации.
Профиль задаётся термами проявлений граней личности, например низким, средним или высоким значением. 
Параметры ситуации и отношения могут задаваться термами или численно, с последующей фаззификацией (с помощью также заданных терм множеств и функций принадлежности).

## Слайд 12. Вычисление коэффициентов для перевзвешивания

Вычисление коэффициентов для перевзвешивания выполняется с помощью правил.
Правило имеет вид: если выполнены условия по профилю, ситуации и отношению к собеседнику, то для заданного д-сценария применяется эффект.
Эффект правила переводится в числовой множитель: от сильного ослабления до сильного усиления. 
Если для сценария сработало несколько правил, их влияние объединяется через взвешенное среднее.
Пример правила также представлен на слайде.
